# NightBar Club 酒吧点单小程序

一个基于HTML、CSS和JavaScript的酒吧点单小程序界面设计。

## 特性

- 🌙 暗色主题设计
- 🔮 玻璃拟态效果
- 📱 响应式设计（iPhone 16 Pro Max优化）
- 🎨 星巴克风格UI设计
- 🖼️ 本地图片资源

## 页面功能

- **首页**: 展示酒吧介绍、会员权益、快捷入口
- **菜单页**: 商品浏览，支持列表视图和瀑布流视图切换
- **购物车**: 商品管理和结算
- **订单进度**: 实时查看订单状态
- **个人中心**: 用户信息和功能设置

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- 玻璃拟态设计
- 响应式布局
